# Notes for Claude

## How to write to me

**Always write in extremely simple English.**

This is a permanent rule. It applies to every answer, every session, every
topic. It covers chat replies, code comments, commit messages, and pull
request text.

How to do it:

- Use short sentences. One idea per sentence.
- Use easy, everyday words.
- No jargon. If a technical word is needed, say what it means right after it.
- Give the answer first. Explain why after.
- Go step by step.
- Keep it short. Cut any word that is not needed.
- Use plain lists, not long paragraphs.
